### Encrypting programm

A simple ruby script that encrypt 
user input using one of two encrypting 
algorithms

### Usage

```sh
$ ruby main.rb

Введите слово или фразу для шифрования:
Top secret USA data: bla bla bla
Каким способом зашифровать:
1. MD5
2. SHA1
1
Вот что получилось:
1c867c0c35c7ed49d23ddbe10e77d8eb
```
